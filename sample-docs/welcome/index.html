欢迎使用 docsite

这是一篇示例文档。docsite 把这样的 HTML 文档组织成可分类、可搜索的站点，给团队通过网址访问。

文档长什么样

每篇文档是一个目录，包含入口 HTML 和它引用的全部图片：

welcome/
├── index.html      ← 你正在看的入口
└── .docmeta.yaml   ← 标题/分类/日期（可选，不写也能自动推断）

发布一篇文档

  1. 把文档目录放进 site/docs/
  2. 运行 python3 build.py

站点首页、分类页、搜索页会自动更新。

跨目录引用的文档怎么办

如果你的 HTML 里有 src="../别的目录/图片.png" 这种跨目录引用，先用 publish_doc.py 打包成自包含：

python3 publish_doc.py path/to/your-doc
# 产物在 dist/your-doc/，所有图片已收进 assets/，路径已重写
